# C++ STL

## vector
* vector v(n) -> 기본값으로 초기화된 n개의 원소
* v.assign(n,x) -> v에 x값으로 n개의 원소 할당
* v.at(i) -> v의 i번째 원소 참조
* v.front() -> v의 처음 원소
* v.back() -> v의 마지막 원소
* v.empty() -> v가 비었는지 조사
* v.push_back(x) -> v의 끝에 x를 추가

## change type
* int -> string : to_string()
* string -> int : stoi()
* char* -> int : atoi()
